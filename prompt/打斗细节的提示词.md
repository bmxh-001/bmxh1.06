# AI小说创作：填充打斗细节的提示词

本文档提供一系列提示词，旨在帮助AI在小说创作过程中，通过特定的打斗公式，更生动地填充打斗场景的细节。

## 一、核心打斗公式：

请围绕以下五个核心要素，细致描绘角色的每一次出手：

1.  **神态与心理动作描写：** 角色在出招前、出招时、出招后的神态变化，以及内心活动（如决绝、犹豫、愤怒、冷静等）。
2.  **出手时的音效：** 描述招式发动时产生的声音，例如破空声、金属碰撞声、能量爆鸣声、衣袂撕裂声等。
3.  **出手造成的攻击特效：** 描述招式本身产生的视觉效果，例如剑光、拳风、法术光芒、气浪等。
4.  **对手眼中，出手造成的攻击特效：** 从对手的视角描述攻击的威势和视觉冲击，强调其感受到的压迫感或惊异。
5.  **出手造成的伤害描述：** 描述招式命中目标后造成的具体伤害效果，例如伤口情况、对手的反应（痛哼、后退、倒地等）、环境破坏等。

## 二、具体应用提示

### 提示词结构模板：

“请为角色「[角色名A]」设计一个攻击动作，目标是「[角色名B/物体]」。

1.  **神态与心理：** 在「[角色名A]」[准备/发动/完成]攻击时，其神态是「[具体神态，如：眼神冰冷、嘴角勾起一抹残忍的笑意、面色凝重]」，内心「[具体心理活动，如：充满了杀意、在计算最佳攻击时机、感到一丝不安但仍旧出手]」。
2.  **音效：** 当「[角色名A]」的「[招式名/攻击方式，如：铁拳、长剑、法术]」出手时，发出了「[具体音效，如：沉闷的破风声、尖锐的剑鸣、能量汇聚的嗡鸣]」。
3.  **攻击特效（自身视角）：** 「[角色名A]」的「[招式名/攻击方式]」在空中划出「[具体轨迹/形态，如：一道耀眼的白光、一团燃烧的火焰、无数细密的风刃]」，携带着「[具体气势，如：雷霆万钧之力、撕裂一切的锋锐、冰冷刺骨的寒意]」。
4.  **攻击特效（对手视角）：** 在「[角色名B]」的眼中，「[角色名A]」的攻击如同「[具体比喻，如：一道不可抵挡的洪流、死神的镰刀、从天而降的审判之光]」，让他/她感到「[具体感受，如：窒息般的压迫、毛骨悚然的恐惧、无处可逃的绝望]」。
5.  **伤害描述：** 「[招式名/攻击方式]」[击中/擦过/被格挡]「[角色名B]」的「[具体部位/武器]」后，「[角色名B]」[具体反应，如：闷哼一声，嘴角溢出鲜血、踉跄后退数步、兵器上出现裂痕、被击飞出去]，其「[受伤部位的具体描述，如：胸口出现一道深可见骨的伤痕、手臂被震得发麻、护体罡气瞬间破碎]」。同时，[周围环境可能发生的变化，如：地面龟裂、气浪吹飞了附近的杂物]。

请基于此模板，详细描写该攻击场景。”

### 示例：

“请为角色「剑客李青」设计一个攻击动作，目标是「魔教长老」。

1.  **神态与心理：** 在「剑客李青」发动攻击时，其神态是「眼神锐利如鹰隼，表情冷峻」，内心「充满了对邪魔外道的憎恶与必杀的决心」。
2.  **音效：** 当「剑客李青」的「青锋剑」出手时，发出了「一声清越的龙吟般的剑鸣，以及剑锋高速切割空气产生的‘咻咻’声」。
3.  **攻击特效（自身视角）：** 「剑客李青」的「青锋剑」在空中划出「一道迅疾如闪电的青色剑芒」，携带着「无匹的锋锐之气，仿佛能斩断世间一切」。
4.  **攻击特效（对手视角）：** 在「魔教长老」的眼中，「剑客李青」的攻击如同「一道从九天之上降下的青色雷霆」，让他感到「一股死亡的阴影笼罩心头，避无可避」。
5.  **伤害描述：** 「青锋剑」的剑芒掠过「魔教长老」的「左肩」后，「魔教长老」惨叫一声，左臂瞬间被齐肩斩断，鲜血如喷泉般涌出，他踉跄后退，脸上充满了难以置信的惊恐。同时，他身后的石壁被逸散的剑气划出数道深深的剑痕。

请基于此模板，详细描写该攻击场景。”

## 三、连续打斗与情境定制

*   **指令：** “在接下来的[场景描述，如：狭窄的巷道中/华丽的宫殿内/广阔的草原上]，「[角色A]」与「[角色B]」展开激烈搏杀。请运用上述五要素打斗公式，为他们设计至少三轮攻防转换，并注意通过神态、心理、音效、特效和伤害描述的递进，展现战斗的激烈程度和双方状态的变化。”
*   **指令：** “针对以下人物设定：[角色A小传，包含武功特点和性格] 和 [角色B小传，包含武功特点和性格]，以及当前情境：[情境描述，如：突袭、遭遇战、生死决斗]。请创作一段他们之间的打斗场面，要求不仅要打斗激烈，还要通过打斗细节鲜明地展现其性格特点和武功路数。”

希望这些提示词能帮助AI创作出更加精彩纷呈、拳拳到肉的打斗场面！
