# AI小说创作：丰富对话细节的提示词

本文档提供一系列提示词，旨在帮助AI在小说创作过程中，通过“常用人物 + 动作/神态 + 语言”的公式，更生动地填充对话细节。

## 一、核心公式：人物 + 行为表现 + 语言

*   **基本指令：** “为角色「[角色名]」设计一段对话。在说出「[对话内容]」时，请结合其「[具体动作或习惯性小动作，如：轻敲手指、整理衣领]」和「[具体神态，如：微笑、皱眉、若有所思]」进行描写，使对话更具表现力。”

## 二、具体应用提示

### 1. 人物 + 动作 + 语言

*   **指令：** “角色「[角色名]」一边「[做出某个动作，如：踱步、擦拭武器、倒茶]」，一边说道：「[对话内容]」。请描写此场景。”
*   **示例：** “角色「李将军」一边擦拭着他的佩剑，一边沉声说道：「明日之战，许胜不许败。」”

### 2. 人物 + 神态 + 语言

*   **指令：** “角色「[角色名]」脸上露出「[某种神态，如：不屑的冷笑、温暖的笑容、凝重的表情]」，然后说道：「[对话内容]」。请描写此场景。”
*   **示例：** “角色「王姑娘」脸上露出俏皮的笑容，歪着头说道：「你猜？」”

### 3. 人物 + 动作 + 神态 + 语言

*   **指令：** “角色「[角色名]」先是「[做出某个动作]」，接着「[展现某种神态]」，然后才开口说道：「[对话内容]」。请详细描写这个过程。”
*   **示例：** “角色「老管家」先是恭敬地鞠了一躬，脸上带着谦卑的微笑，然后才开口说道：「少爷，老爷请您过去一趟。」”

### 4. 强调特定情绪的对话

*   **指令：** “角色「[角色名]」此刻「[非常愤怒/悲伤/喜悦/紧张等]」。他/她「[结合情绪的动作，如：猛地站起、声音颤抖、手舞足蹈、坐立不安]」，「[结合情绪的神态，如：怒目圆睁、泪流满面、笑容灿烂、眼神躲闪]」，然后「[带着强烈情绪]」说道：「[对话内容]」。”
*   **示例：** “角色「张三」此刻非常愤怒。他猛地一拍桌子，双眼因愤怒而赤红，额头上青筋暴起，然后咆哮道：「你们欺人太甚！」”

### 5. 暗示潜台词的对话

*   **指令：** “角色「[角色名]」说出「[对话内容]」时，其「[细微的动作或眼神变化，如：眼神飘忽、手指无意识地蜷缩、嘴角微妙的抽动]」暗示了其内心的「[真实想法或未说出口的话]」。请进行描写。”
*   **示例：** “角色「丞相」微笑着说「一切安好」时，他端起茶杯的手指却几不可察地收紧了一下，眼神深处闪过一丝不易察觉的忧虑，暗示了他内心的不安。”

## 三、情境定制与连续对话

*   **指令：** “在接下来的[场景描述，如：谈判桌上/密室之中/战场之上]中，角色「[角色A]」和「[角色B]」将进行一段关于「[对话主题]」的对话。请运用「人物+动作/神态+语言」的公式，为他们设计三轮以上的对话，并注意通过动作和神态的变化来展现他们情绪和态度的转变。”
*   **指令：** “针对以下人物设定：[角色A小传] 和 [角色B小传]，以及当前情境：[情境描述]。请创作一段他们之间的对话，要求对话不仅要推动情节，还要通过各自的动作和神态鲜明地展现其性格特点。”

希望这些提示词能帮助AI创作出更加生动、富有层次感的对话场景！
