# AI小说创作：丰富人物描写的提示词

本文档提供一系列提示词，旨在帮助AI在小说创作过程中，通过“人物 + 外貌/神态/动作 + 比喻/象征/夸张”的公式，更生动地刻画人物。

## 一、核心公式：人物 + 细节描写 + 修辞手法

*   **基本指令：** “针对角色「[角色名]」的「[外貌/神态/动作]」，运用「[比喻/象征/夸张]」的公式，生成一段富有表现力的人物描写。”

## 二、具体应用提示

### 1. 人物 + 外貌 + 比喻

*   **指令：** “描写角色「[角色名]」的「[具体外貌特征，如：眼睛、头发、身材、伤疤等]」，请用一个生动的「比喻」来形容其特点。例如：‘他的眼神像鹰一样锐利’或‘她的笑容如春日阳光般温暖’。”

### 2. 人物 + 神态 + 比喻

*   **指令：** “角色「[角色名]」此刻的神态是「[具体神态，如：专注、迷茫、坚定、哀伤等]」，请用一个贴切的「比喻」来描绘这种神态。例如：‘他专注的神情，仿佛一位正在雕琢稀世珍宝的工匠’。”

### 3. 人物 + 动作 + 比喻

*   **指令：** “描写角色「[角色名]」正在进行的「[具体动作，如：行走、奔跑、挥手、思考时的习惯动作等]」，请用「比喻」使其更加形象。例如：‘他走路的姿势像松树一样挺拔’或‘她跳舞时轻盈得如同一只蝴蝶’。”

### 4. 人物 + 外貌 + 象征

*   **指令：** “角色「[角色名]」的「[具体外貌特征]」具有某种「象征意义」，请结合其性格或命运进行描写。例如：‘他额头上的那道疤痕，是他不屈意志的象征’。”

### 5. 人物 + 神态 + 象征

*   **指令：** “角色「[角色名]」的「[具体神态]」象征着「[某种内在品质或预示]」，请进行富有深意的描写。例如：‘她平静的眼神中，象征着暴风雨前的宁静’。”

### 6. 人物 + 动作 + 象征

*   **指令：** “角色「[角色名]」的「[特定动作或习惯]」象征着「[其性格特点或命运走向]」，请进行解读式描写。例如：‘他反复摩擦手指的动作，象征着他内心的不安与犹豫’。”

### 7. 人物 + 外貌 + 夸张

*   **指令：** “为了突出角色「[角色名]」的「[具体外貌特征]」，请使用「夸张」的修辞手法。例如：‘他的鼻子简直可以和匹诺曹媲美’或‘她的头发像瀑布一样垂到腰际’。”

### 8. 人物 + 神态 + 夸张

*   **指令：** “角色「[角色名]」的「[具体神态]」表现得极为「[某种极端情绪或状态]」，请用「夸张」手法加以强调。例如：‘他惊讶得下巴都快掉到地上了’。”

### 9. 人物 + 动作 + 夸张

*   **指令：** “角色「[角色名]」的「[具体动作]」幅度极大或效果显著，请用「夸张」手法进行描绘。例如：‘他一拳打出，力量大得仿佛能撼动山岳’。”

## 三、综合运用与情境定制

*   **指令：** “在接下来的[小说章节/特定场景]中，当角色[角色名]出场或进行[关键行为]时，请综合运用「人物+外貌/神态/动作+比喻/象征/夸张」中的至少两种组合，进行[字数]左右的细节描写，重点突出其[性格特征/当时状态]。”
*   **指令：** “针对以下人物设定：[在此处插入具体人物小传，包括性格、背景、以及期望通过外貌/神态/动作表达的特质]，请创作一段运用了「[指定细节，如：眼神] + [指定修辞，如：象征]」的人物描写。”

希望这些提示词能帮助AI创作出更加丰满、立体的人物形象！
